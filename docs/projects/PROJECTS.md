## How to Use

```bash
# Create a new project
pm-project new "Project Name"

# Add opportunities to a project
pm-project add "Project Name" OPP-001

# Set focus on a project
pm-project focus "Project Name"

# View all projects
pm-project list
```

## Project Lifecycle

1. **Create** - Define project goal and success metrics
2. **Populate** - Add opportunities from the backlog
3. **Focus** - Set as the active project for the team
4. **Execute** - Move opportunities through stages
5. **Archive** - Complete and archive when done
